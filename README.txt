Для развёртывания понадобится nodejs 16+ версии

В папке client лежит React проект
В папке server лежит Nodejs, на базе фреймворка express

Для работы сервера создайте базу данных mysql

Создание БД:
    1. Запустите OpenServer
    2. Перейдите в PhpMyAdmin
    3. Создайте БД
    4. Импортируйте файл таблицы messages

Настройка:
    1. В папке server, создайте файл .env и вставьте туда текст ->
    PORT=8000
    DB_NAME= Название вашей бд
    DB_USER= Имя
    DB_PASSWORD= Пароль
    DB_HOST= localhost
    
    2. В терминале перейди в папку client и пропишите - npm install (установит все пакеты)
    3. То же самое проделайте в папке server
    4. Далее в папке server пропишите - npm run dev  (запустит как клиент, так и сервер)
    5. Далее откройте браузер по url - http://localhost:3000
